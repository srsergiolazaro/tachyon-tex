Moonshot Performance

We identified that the primary bottleneck in LaTeX compilation is the System Call Barrier. Tachyon-Tex bypasses it completely.

  • 01

    Embedded Engine

    Standard compilers spawn multiple OS processes. We embed the Tectonic engine as a native library.

  • 02

    Zero-Disk Latency

    All operations happen in RAM. The .tex input and .pdf output never touch the SSD.

  • 03

    Warm Cache

    The Docker image is pre-loaded with common libraries, eliminating cold-start latency.

Quick Start

Launch the engine in seconds using Docker.

docker pull srsergio/tachyon-tex:latest
docker run -p 8080:8080 srsergio/tachyon-tex

Then send your project ZIP via CLI or use the Web UI.

curl -X POST -F "file=@project.zip" http://localhost:8080/compile -o out.pdf
